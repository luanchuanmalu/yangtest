直观的开发过程
SimpleMovingAvgStrategy.m

  classdef SimpleMovingAvgStrategy < KSStrategy

      properties
          pinganPrices
      end

      methods

          function init(obj, context)
              obj.pinganPrices = [];
          end

          function onData(obj, data, context, action)

              price=data.query('pingan','price','double');
              obj.pinganPrices = [obj.pinganPrices price];

              if length(obj.pinganPrices) >= 10
                  movingAverage = mean(obj.pinganPrices(end-9:end));
              else
                  return;
              end

              if price>=movingAverage
                  action.order('pingan', 1);
              elseif price<movingAverage
                  action.order('pingan', -1);
              end

          end

      end

  end
*这是一个简单的单均线策略，请前往文档了解如何快速开发此策略
